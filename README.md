# Software Testing Project - MaiNguyen.vn

## 1. Project Overview
- **Target:** Retail technology website [MaiNguyen.vn](https://www.mainguyen.vn/)
- **Objective:** Perform Manual Testing to ensure the quality and stability of core system features.
- **Role:** Tester (Academic Group Project - Team of 2).

## 2. Key Activities
- **Requirement Analysis:** Analyzed system specifications to define test boundaries.
- **Test Design:** Designed **30 comprehensive Test Cases** covering critical user flows:
    - Product Search functionality.
    - Advanced Filtering (by brand, price range, etc.).
    - Cart Management (adding/removing items, quantity updates).
- **Test Execution:** Performed manual testing and verified results (Expected vs. Actual).

## 3. Techniques & Tools
- **Testing Techniques:** Boundary Value Analysis, Equivalence Partitioning.
- **Documentation:** Microsoft Word, Microsoft Excel.
- **Automation Tools:** Selenium IDE (Basic Record & Playback).
